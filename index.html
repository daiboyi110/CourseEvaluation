Upload one or more course evaluation PDFs to compare mean scores across courses

How to Use This Tool

1. Select Your Course Type

Choose the type that matches your Blue evaluation PDFs before uploading:

  • Sole Instructor / Course Coordinator — evaluations with 21 instructor questions
  • Co-Instructor — evaluations with 12 instructor questions
The course type locks once files are loaded. Click Clear All to switch types.

2. Upload PDF Files

Drag and drop one or more Blue evaluation PDF files onto the upload zone, or click Choose PDF files. Upload multiple PDFs to compare courses side by side. Valid files are displayed even if some fail.

3. Read the Results Table

  • Each column is one course, labeled with its course code and instructor name.
  • Each row is one evaluation question showing the mean score per course.
  • Survey Completion — responses out of enrollment and the completion rate.
  • Teaching Effectiveness Score — average of key teaching questions (see below).

4. Score Color Legend

4.25 – 5.00

Below 3.50

5. Teaching Effectiveness Score

  • Sole Instructor / Course Coordinator: average of Q1, Q4, Q7, Q10, Q16, Q19
  • Co-Instructor: average of Q1, Q3, Q6, Q7, Q10, Q11

6. File Naming Convention

The tool reads the course code and instructor name from the filename automatically. For best results, use the standard Blue export format:

Example: Fall 2024 (KINE 2250) Jane Doe_Section1.pdf displays as KINE 2250 — Jane Doe

7. Other Actions

  • Click Hide Original Question Scores to collapse question rows and show summary rows only.
  • Click × next to any file to remove it — the table updates immediately.
  • Click Clear All to reset everything.
  • All processing happens in your browser — no data is uploaded to any server.
